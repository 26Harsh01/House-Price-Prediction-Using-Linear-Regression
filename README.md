# 🏠 House Price Prediction Using Linear Regression

## 📌 Project Overview

This project demonstrates an end-to-end implementation of **Simple Linear Regression** and **Multiple Linear Regression** to predict house prices using various housing attributes. The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, regression assumption testing, visualization, and model serialization.

---

## 🎯 Objectives

* Understand the concepts of Simple and Multiple Linear Regression.
* Perform data cleaning and preprocessing.
* Explore the dataset using EDA.
* Build predictive regression models.
* Evaluate model performance using standard regression metrics.
* Validate regression assumptions.
* Visualize regression results and feature relationships.

---
